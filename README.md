## 🎓 ¿Qué he aprendido en esta etapa?

En estos proyectos he consolidado los pilares fundamentales del lenguaje:
- **Estructuras de Control:** Uso avanzado de `if`, `for`, y `while`.
- **Colecciones:** Gestión eficiente de `lists`, `dicts` y `tuples`.
- **Modularidad:** Creación de funciones y organización de código en módulos.
- **Persistencia:** Lectura y escritura de archivos (en proyectos como Home Budget).
- **Control de Versiones:** Uso de Git para gestionar el historial de desarrollo.
